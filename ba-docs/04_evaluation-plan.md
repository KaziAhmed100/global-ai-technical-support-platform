# Evaluation Plan (Generic)

## Pilot design
- Select 3–5 high-volume support categories.
- Run 4–8 weeks with baseline comparison.

## Metrics
- Policy-correct answer rate (supervisor-reviewed)
- Time-to-resolution
- Escalation rate
- Reopen rate (ticket reopened after “resolved”)
- User satisfaction (short survey)

## Method
- Use blinded review of sampled conversations.
- Track outcomes by category and role.
- Publish summary results as a reusable evaluation protocol.
